# Brick Game

벽돌 깨기 게임입니다!

---

### 실행 환경
**python 3.7+**  
https://www.python.org/downloads/

**pygame**
```bash
!pip install pygame
```

### 실행 방법

1. 코드를 다운로드합니다.
2. run.py를 실행합니다.

### 조작

| 조작 | 행동 |
|:----------:|:----------:|
| space | 게임 시작 |
| ESC | 게임 종료 |
| KeyLeft | 패들을 왼쪽으로 이동 |
| KeyRight | 패들을 오른쪽으로 이동 |
